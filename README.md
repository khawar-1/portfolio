# Cinematic Canvas | Creative Developer Portfolio

An immersive, high-end portfolio experience built with React, TypeScript, GSAP, and Three.js.

## 🚀 Features

- **Cinematic Preloader:** A custom GSAP sequence with a geometric reveal mask.
- **Immersive 3D Visuals:** WebGL particle systems powered by Three.js for a premium interactive atmosphere.
- **GSAP Animations:** Smooth scroll-driven cinematics, text reveals, and complex layout transitions.
- **Smooth Scrolling:** Integrated with Lenis for a weighty, high-end "Apple-style" scroll feel.
- **Custom Cursor:** Magnetic hover effects and dynamic cursor state transitions.
- **Data-Driven Architecture:** Entire portfolio content manageable via a single structured TypeScript file.
- **Responsive & Performant:** Fully optimized for all device breakpoints and 60fps interaction.

## 🛠️ Tech Stack

- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS & Vanilla CSS
- **Motion:** GSAP (ScrollTrigger, TextPlugin)
- **3D:** Three.js
- **Smooth Scroll:** Lenis
- **Build Tool:** Vite

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run developmental server: `npm run dev`
4. Build for production: `npm run build`
